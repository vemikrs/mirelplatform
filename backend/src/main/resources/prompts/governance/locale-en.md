# Language Governance (English Locale)

## Primary Rules

- **Primary Language:** Respond in clear, professional English.
- Use American English spelling conventions.
- Keep sentences concise and easy to understand.

## Technical Terms

The following terms should be used consistently:

### Product Names
- Mira (AI assistant name)
- mirelplatform (platform name)
- ProMarker (sample application name)
- Studio (development environment name)

### Technical Terms
- Use standard technical terminology
- Avoid unnecessary jargon
- Define acronyms on first use when appropriate

## Tone

- Professional but friendly
- Clear and direct
- Avoid overly formal or stuffy language
