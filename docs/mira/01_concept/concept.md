# Mira Concept

## Overview
Mira（mirel Assist）は、mirelplatform 全体を横断して機能する AI アシスタントである。  
ユーザ操作の習熟、設定作業、エラー理解、ワークフロー構築など、  
プラットフォーム上の複雑性を軽減し、利用者の生産性を最大化することを目的とする。

Mira は以下の3領域を中心に機能する。

1. **汎用的な生成 AI チャット**  
2. **アプリケーション単位のコンテキストヘルプ**  
3. **Studio / Workflow における Assistant Agent**

これらを統合することで、mirelplatform の「操作・理解・設計」を AI が一貫して支援する。

---

## Positioning
Mira は単なるチャットボットではなく、  
mirelplatform の **"Unified Assistance Layer"（統合支援レイヤ）** と位置づける。

- UI の理解を助ける  
- 設定をガイドする  
- エラーを要約し原因を示す  
- 自動化ワークフローの構築を補助する  

これらは、エンタープライズ環境における「複雑さ」そのものを縮減するための設計である。

---

## Scope
Mira の初期バージョン（v0.1）は以下を対象とする。

- mirelplatform 内のアプリ横断情報（画面ID、ロール、操作対象）  
- Studio / Workflow のコンテキスト  
- 設定・エラー説明の支援  
- ロールベースの回答最適化  

中期的には「操作の自動化・設計補助」へ進化させる。

---

## Not in Scope (v0.1)
- 自律実行するオペレーション  
- テナント横断のデータ解析  
- ユーザ固有データの学習  
- 外部 API 連携の自動設定  

以上は v0.2 以降に検討する。
