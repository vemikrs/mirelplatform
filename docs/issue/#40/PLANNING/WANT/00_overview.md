## ⚖️ MUST vs WANT 判断基準

```
MUST（必須）として判断される条件:
  ✅ SaaSとして提供するために法的・技術的に必須
  ✅ 競合サービスが全て提供している標準機能
  ✅ セキュリティ・コンプライアンス上必須
  ✅ サービス運用に不可欠

WANT（望ましい）として判断される条件:
  ⭐ 競争優位性を生む差別化機能
  ⭐ 顧客満足度向上に寄与
  ⭐ 特定セグメント向け機能
  ⭐ 将来的な収益拡大に貢献
```
